## firstinstall
`./deploy_files`

## afterinstall

### show zshrc list
`zshconfig list`

### edit zshrc
`zshconfig #{configname}`

### reload zshrc
`zshload`

### edit vimrc
`vimrc edit #{vimrcname}`

### move dotfiles directory
`dot move`

### update dotfile to github repository
`dot update`

### deploy all dotfiles
`dot deploy`

### install rubygems
`dot bundle`

### install pip modules
`dot pip`
